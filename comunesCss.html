ESTILOS MÁS COMUNES PARA CSS

Algunos de los elementos más comunes en CSS son:

Selector de elemento: Se utiliza para seleccionar todos los elementos de un tipo específico. Ejemplo:

h1 {
color: blue;
}

Selector de clase: Se utiliza para seleccionar elementos que tienen un atributo de clase específico. Ejemplo:

.destacado {
font-weight: bold;
}

Selector de ID: Se utiliza para seleccionar un elemento que tiene un atributo de ID específico. Ejemplo:

#encabezado {
background-color: grey;
}

Pseudo-selector: Se utiliza para seleccionar elementos basados en su estado o posición en relación con otros elementos. Ejemplo:

a:hover {
color: red;
}

Propiedades de estilo: Se utilizan para definir el aspecto de los elementos seleccionados. Ejemplo:

color: blue;
font-size: 16px;
background-color: yellow;

Estos son solo algunos de los principales estilos y elementos en CSS, hay muchos más que se pueden utilizar para personalizar la apariencia de una página web.
