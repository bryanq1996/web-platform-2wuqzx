ESTÁNDARES HTML

HyperText Markup Languageo HTML por sus siglas en inglés es el lenguaje de marcado estándar utilizado para crear páginas web. Los estándares HTML se refieren a las especificaciones técnicas que definen cómo debe escribirse el código HTML para que los navegadores web puedan interpretarlo correctamente y mostrar la página web de manera consistente.

El World Wide Web Consortium (W3C) es la organización encargada de desarrollar y mantener los estándares HTML. El primer estándar HTML, conocido como HTML 2.0, fue publicado en 1995, y desde entonces ha habido varias revisiones y versiones nuevas.

La versión más reciente de HTML es HTML5, que se lanzó en 2014. HTML5 introdujo muchas nuevas características y mejoras, incluyendo soporte para multimedia, gráficos vectoriales, almacenamiento local, formularios avanzados y más.

Los estándares HTML también especifican cómo debe estructurarse y organizarse el contenido de una página web, utilizando elementos HTML como encabezados, párrafos, listas y enlaces. Además, los estándares incluyen directrices para el uso de estilos CSS (Cascading Style Sheets) y scripts JavaScript en las páginas web.

Es importante seguir los estándares HTML al crear páginas web para garantizar que sean accesibles para todos los usuarios, incluyendo aquellos que utilizan tecnologías de asistencia como lectores de pantalla, y para que los navegadores web puedan mostrar la página de manera coherente en diferentes dispositivos y plataformas.

ESTRUCTURA DE UNA PÁGINA WEB

Una página web típica consta de tres lenguajes clave: HTML, CSS y JS.

HTML (HyperText Markup Language) es el lenguaje de marcado que se utiliza para definir la estructura y el contenido de una página web. Proporciona etiquetas que se utilizan para definir los elementos de una página, como encabezados, párrafos, imágenes, enlaces y formularios.

CSS (Cascading Style Sheets) es el lenguaje que se utiliza para definir el aspecto visual de una página web. Se utiliza para controlar la presentación de los elementos HTML, como el tamaño y el color del texto, el diseño de la página, los colores de fondo, etc.

JS (JavaScript) es un lenguaje de programación que se utiliza para crear interacciones dinámicas en una página web. Se utiliza para agregar funcionalidades como animaciones, efectos visuales, validación de formularios, y para realizar solicitudes de red para interactuar con bases de datos y API.

En conjunto, HTML, CSS y JS permiten la creación de páginas web interactivas y atractivas con una estructura clara y una apariencia atractiva.

PRINCIPALES COMPONENTES/ELEMENTOS (HTML)

HTML (Hypertext Markup Language) es el lenguaje de marcado estándar para la creación de páginas web. Los principales componentes o elementos de HTML son los siguientes:

Etiquetas: son los elementos básicos de HTML que se utilizan para definir el contenido y la estructura de una página web. Las etiquetas tienen una sintaxis que consiste en un nombre de etiqueta rodeado de corchetes angulares, por ejemplo <html> o <body>.

Atributos: son valores que se utilizan para modificar el comportamiento o la apariencia de las etiquetas. Los atributos se añaden a las etiquetas mediante el uso de pares clave-valor, por ejemplo href="https://www.ejemplo.com" en la etiqueta <a>.

Elementos de texto: son los elementos utilizados para mostrar texto en una página web. Algunos ejemplos de elementos de texto son <h1> para encabezados, <p> para párrafos y <a> para enlaces.

Imágenes: las imágenes se insertan en una página web mediante la etiqueta <img>, que utiliza el atributo src para indicar la ubicación de la imagen.
Listas: se utilizan para mostrar elementos en forma de lista, ya sea ordenada (<ol>) o no ordenada (<ul>).

Tablas: se utilizan para organizar datos en filas y columnas. Las tablas se crean mediante las etiquetas <table>, <tr> y <td>.

Formularios: se utilizan para recopilar información del usuario y enviarla al servidor. Los formularios se crean mediante la etiqueta <form> y contienen elementos como <input> y <button>.

Metaetiquetas: son etiquetas especiales que se utilizan para proporcionar información adicional sobre la página web, como el título, la descripción y las palabras clave. Las metaetiquetas se incluyen en la sección <head> de la página web.

PRINCIPALES ESTILOS PARA CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para definir el estilo y la presentación de documentos HTML, XML y otros tipos de documentos web. Los principales estilos para CSS son:

Estilo en línea: Se aplica a un elemento específico utilizando el atributo "style" en la etiqueta HTML. Ejemplo:
<h1> style="color: blue;">Título</h1>
Estilo interno: Se aplica a un documento HTML específico utilizando la etiqueta <style> en la sección de <head>. Ejemplo:

<head>
      <style>
      h1 {
       color: blue;
      }
     </style>
</head>
<body>
    <h1>Título</h1> br </body>
Estilo externo: Se aplica a varios documentos HTML utilizando un archivo CSS externo que se vincula a través de la etiqueta <link> en la sección de <head>. Ejemplo:

<head>
        <link rel="stylesheet" type="text/css" href="estilos.css">
</head>
<body>
        <h1>Título</h1>
